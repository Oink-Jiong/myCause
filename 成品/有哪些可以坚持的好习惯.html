    # 有哪些可以坚持的好习惯

    人之初，性本善；性相近，习相远。人和人的差距更多的来自习惯的不同，看看我20多年来，被生活蹂躏领悟出来的20条好习惯，坚持下来你将贤于翻身。

    # 1、不讲道理

    等等，不是让我们蛮横无理，是沟通不要随便抛大道理，甚至采用教唆、冷嘲热讽的语气，这很可能让问题没有解决的同时，还雪上加霜。想想我们是不是很反感父母说，你还小你啥都不懂，听我们老一辈的...

    # 2、学会倾听

    我们用两年学会说话，却要用一辈子学会闭嘴。——海明威

    说话是大部分人与生俱来的能力，可闭嘴却是许多人终其一生都学不会的技能。会说话是修养，管住嘴是教养。 年少轻狂不懂事，等到去社会大学才领悟到祸从口出的真谛。

    # 3、在给别人提供重要资讯的时候，少用可能也许随便应该等词汇，这会让人感觉你说话不靠谱。

    # 4、不要太在乎别人对自己的评价

    谁人背后无人说，哪个人前不说人？ 因为有时候人更多只是口嗨一样，并不是对你有看法而人身攻击。走自己的路让别人说去吧，大多数人都是路人甲。

    # 5、逢人且说三分话，未可全抛一片心

    与他人交谈时话只能说三分，不能把自己内心的想法全部吐露给别人，要把握好 “边界感” 。

    # 6、不要拍马屁，但要有发自内心赞美别人的好习惯

    国人是很讨厌欺骗的，圆滑的人注定是跳梁小丑。

    # 7、说话可以直率，但不能直接。直率是爽快，直接是目中无人

    # 8、焦虑时多做事

    焦虑大都是因为三思而不行。读书、听课加上思考这些都是没办法最终解决问题的，只有知行合一才能去把问题移除，焦虑才会消除。

    # 9、坚持左右手锻炼

    屏幕前的过儿们，刷久了知乎手指会酸噢，不加锻炼老了有可能关节炎，试试双击屏幕吧

    # 10、多以结果示人，而不是立flag

    不会有太多人关心你的flag，倒不如等你把4、6级或者驾考车证晒到朋友圈来得实在。

    # 11、少做抱怨，特别是越亲近的人

    抱怨没办法解决问题，而且还会让负面情绪传递，倒不如沉心静气询问亲朋好友解决方法。

    # 12、早点考多点证

    不管是驾考还是4、6级专业证书，早考早享受。而且技多不压身，用人单位都是用证书来初步评估个人水平

    # 13、培养自信第一步，每天起床对着镜子自我肯定

    # 14、少喝饮料和奶茶

    其中包含的高糖、咖啡因，不仅会加重痤疮、促进皮脂分泌、加速皮肤糖化和衰老、胰岛素抵抗、肥胖、糖尿病、心血管疾病等等，多喝hot水吧过儿们。

    # 15、避免漫无目的的刷朋友圈、b站、微博、抖音等平台

    在大数据面前，你对自己的了解，可能还没有这些平台的算法能力深入，想要去测试自己的自制力？我想可能没几个坚持下来，我更建议带着一定的需求，找到后及时关闭可能会更好吧，毕竟没人觉得自己时间很多。

    # 16、提升自己的可替代成本

    这点在职场中极为明显，老板裁员看的不是你有多努力，更多情况是你的可替代成本。要是你的可替代成本足够低，20岁就足够迷茫和焦虑了。这是一位摸爬滚打多年的长者告诉我的，你的焦虑更多来自你的可替代成本太低了。

    # 17、按时吃早餐

    不吃早餐老得快，不吃早餐，人体就会动用体内贮存的糖元和蛋白质，久而久之，会导致皮肤干躁、起皱和贫血等，加速人体的衰老。超过9点不吃早餐，会进行肠道重吸收噢，你怕么？

    # 18、保证午休

    确保每天都有15~30分钟的午休，给大脑一个关机休眠时间，下午会精神满满噢。

    # 19、睡觉前不要剧烈运动

    年轻人不讲武德，大学期间还见过一些人，睡觉前还要剧烈运动，嫩得几个相连的床铺排山倒海，这样很影响大家的睡眠质量呢。

    # 20、不要熬夜、不要熬夜、不要熬夜，重要事情说三遍

    熬夜劳累致死的人，在90后大军已经不是两只手脚可以数过来了，白天多提升自己的能力，晚上就少一些焦虑困苦。

    # 21、多做深蹲，多做提肛运动，你迟早会来感谢我，不如现在就双击感谢一下吧。

    自信点，我知道你肯定会感谢我。
