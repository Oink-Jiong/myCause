    # 什么样的男生最受欢迎？

    说真的答应我，如果这个男生是这样的话，请好好珍惜他，因为这已经是国宝了！

    # 1、不一定要有健美的身材，但一定要身体健康！

    毛爷爷在几十年前就告诫我们：身体是革命的本钱。 别趁着年轻，透支自己的本钱，看看微博、抖音上每天有多少猝死的年轻人。要是男生们还时常通宵熬夜吃鸡打王者的话，就不知道明天和猝死哪个先到来，谁希望把未来放置这么多不定因素呢，那就好像每天能睁开眼睛都是赌博胜利一样，如何才能心安。

    要是真的控制不了自己，可以试试这种思想暗示法，想想你游戏结束后，娇妻开着你的法拉利，拿着你几亿资产，还携带几套别墅房产证去改嫁， 这波岂不是巨亏。

    雕雕还是想说，这年代留得青山在，不怕没柴烧，看谁笑到最后。

    # 2、不一定要浪漫，但一定要负责任！

    他们可以不会浪漫，可以是大猪蹄子，因为只要有心，浪漫是可以培养的。

    但是反观如今男生可以足不出户在陌陌、soul等软件遇到一堆红颜知己 （在此希望各位男朋友们放下这些软件，用心对待身边每个人）， 身边莺莺燕燕如此之多，如果不负责任的话，对待另一半甚至是双方家庭可能都会是余生的痛，而且男生可以挥一挥手不带走一片云彩，她可能就是深夜间独自受伤舔舐伤口，默默结痂。

    # 3、不一定要大男子主义，但大事发生一定要拿得主意！

    买房买车等大事，能悉心听取长辈意见的同时，自己拿捏后能拍板。如果没有自己的话语权，到时候娶媳妇难不成也等着父母安排相亲，生多少孩子，啥时候生，都要父母一手操办，那不是巨婴是什么？

    # 4、不一定要挣大钱，但一定要养家！

    其实真的能赚大钱的人并不多，相反不养家的男生并不少

    不养家的男生钱最后到哪里去了，细思极恐，这种男生谁敢亲近，反正雕雕我是怕了，我真的是在现实生活看到不养家人的男性同胞，搞得七八十岁的老翁还要自己出来捡破烂谋生。

    # 5、不一定要开天辟地，但一定要有担当！

    海德格尔在《存在与时间》中曾经说过：“有担当，才有自由。”

    没有担当，特别是那些无所事事吊儿郎当，成天还想着有富婆包养的男生，富婆会喜欢这样的你嘛。喂醒醒了别做梦了，优秀的人这么多，就算是天上掉馅饼，也是腿长跑得快的先抢到。更何况她包养你一时会包养你一辈子吗，可能你被榨干价值后，不久后就换个年轻力壮的了。

    # 6、不一定要经常亲亲抱抱举高高，但要一定不能吝啬自己的赞美！

    言行双管齐下的肯定，你的人缘会更好，脱单率更高噢。就比如看到这里双击屏幕给我一个认可，正能量是会传递的噢。

    # 7、不一定要帅气，但要一定不能邋遢！

    这让我又重新想起我那个宿友，他可以在家里带着几套衣服去学校，然后呢换过的衣服都扔在床尾，等到周末回家再把一星期的衣服，拿回家洗。

    兄弟那是南方大夏天啊，他还天天打篮球，衣服基本都是外衣流到内衣全湿透的，有时候回南天，空气那叫一个酸爽，啊~~ 人是挺帅气的，可是你想象你到他宿舍床位躺下闻闻，你品你细品 ，不行了我要去厕所一回。

    # 8、不一定要能言善道，但一定不要背后说别人坏话！

    喜欢背后说别人坏话，到了网络上可能多半是键盘侠或者大圣母。经常充斥着负能量，四处传递，亦或者是高居道德制高点，对他人进行道德绑架。

    # 9、不一定要温文尔雅，但一定不要粗鲁肤浅！

    之前有个一面之缘的男生不开口则已，一开口就粗鲁肤浅。想想大过年的高朋满座，这个满脸油腻的男生夸夸其谈，不是爆粗话，就是满脸淫笑着乱开黄腔，真的谁敢认这个亲戚，要是在座有人喜欢的话，带走吧，没脸看。

    # 10、不一定要舞剑，但一定手中要有剑！

    这把剑就是你在这个社会上，落脚生存的武器，你不必张牙舞爪，但是一定要让人知道你的底线。 弱肉强食的地球上，你需要培养一个让你基本生存的本领。出来社会不再像学生时代一样，没有人逼你成长，但一定有人让你知道不成长的代价。如果你现在不知道该做什么，好好备考4、6级，考车证，专业技能证，多考几本证书，几年后你会感激我滴！
