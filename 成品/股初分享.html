# 股初分享

大家好，我是一个来着潮汕地区的小朋友，可以叫我雕雕，身边还有人叫我雕神、雕哥、雕兄、雕爷、雕总，都可以大家随便挑一个，哈哈哈

今天我想和大家聊聊“钱”，这个令人又爱又恨，难舍难分的东西

  • # 首先，我先抛出两个问题

1、钱是什么东西？

2、钱是赚来的吗？

大家先思考一下，不用回答就藏在心里（爱我别说出来），下面我带着两个问题，和大家谈天说地，回过头来再对照一下自己的答案

# 那好，相信大家都有自己的答案了，那就进入第一个环节

# 1、钱什么东西？

你可能会笑我问这么傻逼的话题，三岁小孩都知道，人民币就是钱，能用来买东西。

好，原谅我的傻逼，我们稍安勿躁哈。曾经我也被别人提问过，那时候我用一张人民币指着他，大骂那个人傻逼！

但是很快其实我就发现这个回答有瑕疵，那没有人民币之前呢，是铜板？

……以此类推，再搞到秦币，那秦币之前呢？我记得我的体育老师，教我们语文的时候说过，既然可以下定义的东西，怎么可能说改就改呢！钱＝货币真的？

不行我不行了，我直接找了度娘，直接给我这个解释

我问问自己真的懂钱是什么东西吗？不懂钱怎么赚呢？

就像心上人是男是女都不知道，就表白说要处对象，赚到钱成功率多高？更何况我们是想当百万富翁/婆！

也如同来长投学鸡精、学古票，学了后连鸡精是什么，古票是什么都不知道，就想入市

# 下面我来分享一下，我自己深思熟虑以及各方找答案后的结果是：

  • # 钱 ＝ 输出的价值 * 影响力

这个等式可以解释很多现象

1、为啥没有升值加薪，看看是输出苦劳还是功劳，一份功劳胜于十份苦劳！因为这是价值质量的问题！

2、为啥很多人想冒着“巨大风险”创业，因为打工人的价值为谁服务，老板一个人，所以打工人钱少是注定的！因为这是影响力问题！

# 说成人话就是，钱是我们帮助多少人，解决了多大的问题的一种馈赠的工具，而且这个馈赠经常是迟到的！！！

# 好了，很快我们来到了第二个问题

# 2、钱真的是“赚”来的吗？

我不知道是不是我的误解，我总觉得这么些年来，赚钱这个中性词被有些人误解了，包含了赢得、获取、得到甚至是牟取等味道

我个人觉得，最好是要把钱看做一种积极主动输出价值后的馈赠，你就不会总是因为钱来的和预期不匹配，而痛苦不堪，君子求诸己

如果赚钱让你倍感痛苦、焦虑、烦恼，那么你要开始想想你的初心，你从事事业的初心、你赚钱的初心，因为钱只是获得幸福的一种工具！

# 为什么这么说呢？

比方我们长投经常讲起的《富爸爸穷爸爸》，可能很多人想说：哎呀，这本书不就是教我们分清负债和资产吗？

其实我觉得，不然！负债和资产是知识，知识可以学！会计有学吧，但是所有会计师就一定饱含财商吗？个个是百万富翁？

相反，如何跳出“老鼠赛跑”的境地，这才是人生的智慧，智慧只能体验和感悟！就像很多在座小伙伴一样，来长投的目的不单单只是来学习知识，更重要的是《吸引力法则》引领我来到了这里，和一群志同道合积极向上的人，共同实现自己的价值，完成自己的人生目标！

富爸爸传授积极主动、热爱生活、迎难而上思维模式，和穷爸爸只会教自己孩子，忍气吞声的被迫接受自己是穷人的方式完全相反

试问，谁喜欢被命运征服，这样幸福感在哪？穷不一定就不幸福，唉声叹气地看着自己，一步步被逼上不想要的生活而无所作为，才是人生最大的不幸！

再看看如今网络上现实中，每天都有很多人喊没钱，怨天尤人，可是当中有多少人动动手指，百度一下钱是什么东西？赚钱有什么方法？我想应该是投射情绪大于想改变现状吧

因此唯有积极主动才可能进入更好的圈子，我们想想，任何富二代是不是总有一个积极主动、不折不挠的富一代？

这思维模式恰恰和《高效能人士的七个习惯》中第一个习惯相吻合，机会总是留给积极主动的人

人是一种情感动物，是一种潜意识驱动的生物，晓之以情，动之以理，生而为人更多是先动心（情），再动脑（理）。

我把这种潜意识称之为使命，每个人都有自己的使命。就比如《小狗钱钱》一书中的“愿景驱动”；还有稻盛和夫在《心》写到“一切始于心，终于心”

还记得习大大说的吗？不忘初心、牢记使命。心之所向的地方，皆是因为热爱！

# 所以话说回来，怎样才能赚多点钱？

1、找寻自己积极热爱的赛道

2、量力而行，主动解决更大的问题

3、帮助更多的人、群体、社会、国家

# 大道至简，唯有热爱生活并且积极帮助他人的人，才能收获幸福！！！

# 我最后留一个问题，投资古票的收益，到底从哪里来，谁给的？

# 无用之用方为大用，多主动关注国家发生什么，帮助他，与国运同存，你会发现很多价值，不要整天问代码！

# 雕只是传说，嘻嘻，谢谢大家用心的聆听！
