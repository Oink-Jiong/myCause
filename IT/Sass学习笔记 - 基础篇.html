# Sass学习笔记 - 基础篇

# 1. 符号

  • $：变量
  • &：父选择器

# 2. 变量名

  • 命名用中划线（如$highlight-color）

# 3. 混合器

  • @mixin定义、@include引用
  • @mixin默认参数值
    @mixin link-colors(
	$normal,
	$hover: $normal,
	$visited: $normal
  )
{
  color: $normal;
  &:hover { color: $hover; }
  &:visited { color: $visited; }
}

// 调用，$hover和$visited也会被自动赋值为red
@include link-colors(red)
  • @include动态传参，类似js
    • 参数一一对应
      a {
  @include link-colors(blue, red, green);
}
    • $name: value（key，value键值对）
      a {
	@include link-colors(
	  $normal: blue,
	  $visited: green,
	  $hover: red
  );
}

# 4. 继承

  • 属于对象思想
    // 通过选择器继承继承样式
.error {
  border: 1px solid red;
  background-color: #fdd;
}
.seriousError {
  @extend .error;
  border-width: 3px;
}

// .seriousError从.error继承样式，继承父类所有关系，包括深层的选择器
.error a{  // 应用到.seriousError a
  color: red;
  font-weight: 100;
}
h1.error { // 应用到hl.seriousError
  font-size: 1.2rem;
}

# 5. html元素、混合器、继承的命名

  • html元素：是一个有语义的类名，表明了这个html元素的用途，体现html节点结构
  • 继承：继承应该是建立在html元素语义化的关系上，某个类（比如说.seriousError）继承另一个类（比如说.error），体现html节点结构
  • 混合器：是展示性的，它描述了包含它的css规则最终的视觉样式，体现css样式
